## 💻 技术架构规划

### 5.1 技术栈选型

> 项目使用的技术栈、框架版本、核心库。

**⚠️ uniapp 项目强制要求（H5 模式优先）**：

- **开发模式**：**必须使用 H5 模式进行开发和验证**，所有功能必须在浏览器中完整可用
- **启动命令**：通过 `npm run dev:h5` 启动，在浏览器中直接访问和验证
- **验证标准**：H5 模式是唯一的开发和验证标准，禁止在小程序模式下进行开发和验证
- **多端兼容**：如需支持微信小程序等平台，仅使用 uniapp 条件编译语法（`#ifdef MP-WEIXIN`、`#ifndef H5`）添加平台特定代码
- **代码隔离**：条件编译代码应最小化，不得影响 H5 主代码逻辑和可读性
- **禁止事项**：禁止生成小程序相关配置文件和代码，禁止使用 `npm run dev:mp-weixin` 等小程序命令

> ⚠️ **版本说明**：无特殊要求使用以下版本：
>
> - **移动端（uniapp H5）**: Vue 3.4.21, vant4, Vite 5.2.8, Pinia 2.1.7 ，vue-router 4.0.3，axios 1.6.8
> - **Web端（Vue 3）**: Vue 3.4.21, Vite 5.2.0, Element Plus 2.6.1, Vue Router 4.3.0, Pinia 2.1.7，axios 1.6.8
